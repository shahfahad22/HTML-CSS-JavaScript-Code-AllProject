# ElectroMart | Your Trusted Electronics Store

![ElectroMart Logo](./public/logo.png) <!-- Replace with your actual logo or screenshot -->


## Overview

**ElectroMart** is a modern, fully responsive e-commerce website built from scratch.  
It allows users to browse and shop electronics, manage their cart, and navigate a clean multi-page layout.

---

## Features

- **Responsive Design:** Works seamlessly across desktop, tablet, and mobile devices.  
- **Dynamic Product Listing:** Products are rendered dynamically with detailed information.  
- **Shopping Cart Functionality:** Users can add, remove, and update product quantities.  
- **Multi-Page Layout:** Home, About, Contact, and Cart pages for better user experience.  
- **Interactive UI Elements:** Increment/decrement product quantity, cart total calculation, and smooth navigation.  
- **Professional Footer:** Includes popular categories, customer services, and contact information.  

---

## Technologies Used

- HTML5  
- CSS3 (Responsive Layouts)  
- JavaScript (ES6+ Modules & DOM Manipulation)  
- Font Awesome for icons  
- Local Storage for cart persistence
